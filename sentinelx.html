🛡️ SentinelX – Cybersecurity Suite

Role: Lead Developer / Security Architect

Tech Stack: FastAPI, React.js, PostgreSQL, ELK, PyTorch, Kafka, Docker, Kubernetes, Keycloak

Overview

SentinelX is a scalable, modular cybersecurity platform for enterprises, combining threat detection, IAM, encryption, and compliance automation in one unified dashboard.

Key Features

  • 🔐 Identity & Access Management (IAM) with OAuth2, RBAC, LDAP/SSO, and MFA
  • 🧠 AI-powered anomaly detection and GPT-based log analysis assistant
  • 🔍 Kafka-based SIEM + ELK dashboards and real-time log processing
  • 🔒 End-to-end file & message encryption using AES and RSA
  • 📜 Compliance scanner with automated GDPR, SOC2, ISO reports
  • 🛡️ Network threat detection via Suricata/Snort integration
  • ⚙️ SOAR playbooks with integrations (Slack, Jira, AWS)

Impact

  • ✅ Reduced enterprise audit time by 80% with auto-compliance reports
  • ✅ Deployable in cloud, on-prem, or hybrid setups
  • ✅ Modular design allows customization across industries

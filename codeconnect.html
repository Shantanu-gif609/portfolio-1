CodeConnect

Tech Stack: React.js, Next.js, React Native, Firebase, Node.js, OpenAI API, MongoDB, Tailwind CSS, WebRTC, Socket.io

Overview

CodeConnect is a full-stack web and mobile platform that allows students to showcase their projects, receive AI-powered code and resume feedback, and connect with professionals for mentorship, internships, and jobs.

Key Features

  • AI-based project code review and resume scoring using OpenAI
  • Real-time collaboration via video and code sharing using WebRTC + Socket.io
  • Project upload with GitHub integration, tech stack tags, and public profiles
  • Job and internship matching system based on skills and activity
  • Fully synced mobile app built with React Native + Expo

Responsibilities

  • Built frontend with Next.js and Tailwind for responsive UI
  • Developed Node.js/Express backend and integrated OpenAI APIs
  • Configured Firebase for auth, notifications, and storage
  • Implemented WebRTC-based live coding + video rooms

Impact

  • Boosted student visibility to recruiters and mentors
  • Enabled peer learning and collaboration with live tools
